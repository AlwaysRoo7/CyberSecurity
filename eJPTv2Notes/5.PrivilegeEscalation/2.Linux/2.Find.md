## Find World-Writable Files Excluding Symlinks (Privesc)
```text
find / -not -type l -perm -o+w
```
---
## Find the File Named "flag1.txt" in the Current Directory
```text
find . -name flag1.txt
```
---
## Find the File Named "flag1.txt" in the /home Directory
```text
find /home -name flag1.txt
```
---
## Find the Directory Named "config" Under "/"
```text
find / -type d -name config
```
---
## Find Files with the 777 Permissions
```text
find / -type f -perm 0777
```
---
## Find Executable Files
```text
find / -perm a=x
```
---
## Find All Files for User "frank" Under "/home"
```text
find /home -user frank
```
---
## Find Files that Were Modified in the Last 10 Days
```text
find / -mtime 10
```
---
## Find Files that Were Accessed in the Last 10 Days
```text
find / -atime 10
```
---
## Find Files Changed Within the Last Hour (60 Minutes)
```text
find / -cmin -60
```
---
## Find Files Accessed Within the Last Hour (60 Minutes)
```text
find / -amin -60
```
---
## Find Files with a 50 MB Size
```text
find / -size 50M
```
---
## Find Files with the SUID Bit (Allows Running as a Higher Privilege Level)
```text
find / -perm -u=s -type f 2>/dev/null
```
