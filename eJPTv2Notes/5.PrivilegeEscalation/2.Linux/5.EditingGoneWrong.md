## <----- SUID Binaries Enumeration ----->

## Find All SUID Files Owned by Root (Potential Privilege Escalation Binaries)
```text
find / -user root -perm -4000 -exec ls -ldb {} \;
```
---
## Check Allowed Sudo Commands for Current User
```text
sudo -l
```
---
## Found Manual Page of ls with Sudo Privileges
```text
sudo man ls
```
