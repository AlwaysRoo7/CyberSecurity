## To Load the WMAP Extension
```text
msf6> load wmap
```
---
## Add the Site IP (use -h for help)
```text
msf6> wmap_sites -a <IP>
```
---
## Add the Target IP Address as a Target Site
```text
msf6> wmap_targets -t http://<IP>
```
---
## View the Available Sites
```text
msf6> wmap_sites -l
```
---
## List of Defined Targets
```text
msf6> wmap_targets -l
```
---
## Show the Modules that Will Be Used
```text
msf6> wmap_run -t
```
---
## Execute the Scan
```text
msf6> wmap_run -e
```
---
## List Discovered Vulnerabilities
```text
msf6> wmap_vulns -l
```
---
## View Discovered Hosts
```text
msf6> hosts
```
---
## View Discovered Services
```text
msf6> services
```
---
## About WMAP
WMAP is a powerful, feature-rich web application vulnerability scanner that can be used to automate web server enumeration and scan web applications for vulnerabilities. WMAP is available as an MSF plugin and can be loaded directly into MSF. WMAP is fully integrated with MSF, which consequently allows us to perform web app vulnerability scanning from within the MSF.
