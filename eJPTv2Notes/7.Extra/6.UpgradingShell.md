<----- Upgrading Non-Interactive Shells ----->

## Spawn Interactive Bash Shell (Basic Upgrade)
```text
/bin/bash -i
```
---
## Python3 Version of TTY Spawn
```text
python3 -c 'import pty; pty.spawn("/bin/bash")'
```
---
## Spawn Fully Interactive TTY Using Python
```text
python -c 'import pty; pty.spawn("/bin/bash")'
```
---
## Set Terminal Type for Better Shell Behavior
```text
export TERM=xterm
```
---
## Background Shell to Adjust Local Terminal
```text
Ctrl + Z
```
---
## Bring Shell Back to Foreground
```text
fg
```
---
## Reset Terminal Display
```text
reset
```
---
## Check Terminal Type
```text
echo $TERM
```
