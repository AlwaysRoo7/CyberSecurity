## Display Nikto Help Menu and Options
```text
nikto -Help
```
---
## Scan Target Web Server for Vulnerabilities
```text
nikto -h http://<T_IP>
```
---
## Scan Specific Page for File Inclusion Vulnerabilities
```text
nikto -h "http://<T_IP>/index.php?page=arbitrary-file-inclusion.php" -Tuning 5 -Display V
```
---
## Save Nikto Scan Results in HTML Format
```text
nikto -h "http://<T_IP>/index.php?page=arbitrary-file-inclusion.php" -Tuning 5 -o nikto.html -Format htm
```
---
## Scan Target on Specific Port
```text
nikto -h http://<T_IP> -p 8080
```
---
## Scan HTTPS Web Server
```text
nikto -h https://<T_IP>
```
---
## Display Verbose Scan Output
```text
nikto -h http://<T_IP> -Display V
```
---
## Scan Only Interesting Files
```text
nikto -h http://<T_IP> -Tuning 4
```
---
## Scan Only File Upload Vulnerabilities
```text
nikto -h http://<T_IP> -Tuning 5
```
---
## Scan Multiple Vulnerability Categories
```text
nikto -h http://<T_IP> -Tuning 1,2,3,5
```
---
## Ignore SSL Certificate Errors
```text
nikto -h https://<T_IP> -ssl
```
---
## Use Custom User-Agent Header
```text
nikto -h http://<T_IP> -useragent "Mozilla/5.0"
```
---
## Save Output in TXT Format
```text
nikto -h http://<T_IP> -o results.txt -Format txt
```
---
## Save Output in CSV Format
```text
nikto -h http://<T_IP> -o results.csv -Format csv
```
---
## Save Output in XML Format
```text
nikto -h http://<T_IP> -o results.xml -Format xml
```
---
## Scan with Authentication Credentials
```text
nikto -h http://<T_IP> -id admin:password
```
---
## Use Cookies During Scan
```text
nikto -h http://<T_IP> -C all
```
---
## Check Specific CGI Directories
```text
nikto -h http://<T_IP> -Plugins "cgi"
```
---
## Evasion Technique to Bypass Simple Filters
```text
nikto -h http://<T_IP> -evasion 1
```
---
## Scan Through Proxy
```text
nikto -h http://<T_IP> -useproxy http://127.0.0.1:8080
```
---
## Update Nikto Databases
```text
nikto -update
```
