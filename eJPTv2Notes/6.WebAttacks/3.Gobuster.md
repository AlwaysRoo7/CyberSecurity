## Show Gobuster Directory Mode Help Menu
```text
gobuster dir --help
```
---
## Perform Basic Directory Enumeration
```text
gobuster dir -u http://<T_IP> -w /usr/share/wordlists/dirb/common.txt
```
---
## Ignore 403 and 404 Status Codes
```text
gobuster dir -u http://<T_IP> -w /usr/share/wordlists/dirb/common.txt -b 403,404
```
---
## Enumerate Directories and Files Inside /data Path
```text
gobuster dir -u http://<T_IP>/data -w /usr/share/wordlists/dirb/common.txt -x php,xml,txt -r
```
---
## Basic Directory Bruteforce
```text
gobuster dir -u <ip>/content/data/ -w /usr/share/wordlists/dirb/common.txt
```
---
## With File Extensions (php, txt, html, bak, etc.)
```text
gobuster dir -u <ip>/ -w /usr/share/wordlists/dirb/common.txt -x php,txt,html,js,css,bak,backup,old,zip,tar,gz,sql,env,json,xml
```
---
## Using Bigger Wordlist
```text
gobuster dir -u <ip>/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt
```
---
## With Threading (Faster)
```text
gobuster dir -u <ip>/ -w /usr/share/wordlists/dirb/common.txt -t 50
```
---
## Appending File Extensions to Each Word, Using 50 Threads to Find Hidden Files Like Flags, Backups, Configs, or Logs
```text
gobuster dir -u <ip>/sub-dir/ -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,txt,bak,zip,conf,log -t 50
```

## About Gobuster
Gobuster is an open-source tool written in Go used to brute-force directories and files in websites, DNS subdomains, virtual host names on target web servers, and open Amazon S3 buckets.
