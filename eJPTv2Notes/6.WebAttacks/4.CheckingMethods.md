## Get Webpage Content
```text
curl -X GET http://<T_IP>
```
---
## View HTTP Response Headers
```text
curl -I http://<T_IP>
```
---
## Check Allowed HTTP Methods
```text
curl -X OPTIONS http://<T_IP> -v
```
---
## Test POST Request Support
```text
curl -X POST http://<T_IP>
```
---
## Test PUT Request Support
```text
curl -X PUT http://<T_IP>
```
---
## Check Methods on Login Page
```text
curl -X OPTIONS http://<T_IP>/login.php -v
```
---
## Send Empty POST Request to Login Page
```text
curl -X POST http://<T_IP>/login.php
```
---
## Submit Login Form Credentials
```text
curl -X POST http://<T_IP>/login.php -d "name=john&password=password" -v
```
---
## Check Methods on post.php
```text
curl -X OPTIONS http://<T_IP>/post.php -v
```
---
## Check Methods on Uploads Directory
```text
curl -X OPTIONS http://<T_IP>/uploads/
```
---
## Verbose Check on Uploads Directory
```text
curl -X OPTIONS http://<T_IP>/uploads/ -v
```
---
## Create a Sample Text File
```text
echo "Hello World" > hello.txt
```
---
## Upload File to Server
```text
curl http://<T_IP>/uploads/ --upload-file hello.txt
```
---
## Delete Uploaded File
```text
curl -X DELETE http://<T_IP>/uploads/hello.txt
```
