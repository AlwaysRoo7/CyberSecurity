## SNMP Service Detection

```bash
nmap -sU -sV -p 161 <T_IP>
```

## Find SNMP Community String

```bash
nmap -sU -p 161 --script=snmp-brute <T_IP>
```

## Run All SNMP NSE Scripts

```bash
nmap -sU -p 161 --script snmp-* <T_IP> > snmp_info.txt
```
