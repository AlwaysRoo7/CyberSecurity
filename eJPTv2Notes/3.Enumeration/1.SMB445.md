## Nmap SMB Enumeration

### Enumerate SMB Protocol Versions

```bash
sudo nmap -p445 --script smb-protocols <IP>
```

### Display SMB Security Mode Information

```bash
sudo nmap -p445 --script smb-security-mode <IP>
```

---

## smbclient / SMB Tools

### List Available Shares (Anonymous)

```bash
smbclient -L //<T_IP> -N
```

### Connect to Specific Share

```bash
smbclient //<T_IP>/<ShareName> -N
```

### List SMB Shares Using Credentials

```bash
smbclient -L //<T_IP>/ -U <USER>
```

### Access SMB Share Using Credentials

```bash
smbclient //<T_IP>/<ShareName> -U <USER>
```

### Connect with Username & Password

```bash
smbclient //<T_IP>/<ShareName> -U <username>%<password>
```

## 🗺️ SMBMap

### List Shares Without Credentials

```bash
smbmap -H <T_IP>
```

### List Shares With Credentials

```bash
smbmap -H <T_IP> -u <username> -p <password>
```

### Recursive Directory Listing

```bash
smbmap -H <T_IP> -R
```

### Download Specific File

```bash
smbmap -H <T_IP> --download '<path>\<passwords.txt>'
```

### Execute Command (If Permitted)

```bash
smbmap -H <T_IP> -u <admin> -p <password> -x 'ipconfig'
```
