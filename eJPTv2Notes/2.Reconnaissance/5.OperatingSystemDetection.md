## Aggressive Scan (`-A`)

Includes OS detection (`-O`), service/version detection (`-sV`), default NSE scripts (`-sC`), and traceroute.

```bash
nmap -A <T_IP>
```

---

## OS Detection with OS Guessing

Performs a SYN scan, OS detection, and displays possible OS matches with their accuracy percentages.

```bash
nmap -T4 -sS -O --osscan-guess -p- 192.168.10.1
```

---

## OS Detection with Default NSE Scripts

Performs OS detection and executes Nmap's default NSE scripts.

```bash
nmap -sS -O -sC 192.168.10.1
```
